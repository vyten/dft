# dft
## Distans Fart Tid
**En enkel websida för beräkningar.**

Sidan är för att jag skall prova på php, så det är inte den *snyggaste* koden.

#### Distans
(Fart * Tid) / 60 = Distans 
#### Fart
(Distans / Tid) * 60 = Fart
#### Tid
(Distans / Fart) * 60 = Tid
